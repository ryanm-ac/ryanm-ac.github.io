Jan 15, 2017

This website was built using a serverless architecture. Static content is generated by Hugo and hosted on GitHub Pages. Terminal commands are executed in the cloud by AWS Lambda through Amazon API Gateway. Emails are delivered by AWS Simple Email Service (SES).

Other open source technologies used to create this site include:

  • jQuery and jQuery Terminal Emulator
  • Nix - a Hugo theme
  • Bootstrap
  • Google Fonts and Font Awesome
  • MacVim
  • Node.js

Please use the email command from the /home page to send any questions or comments.

Thanks for visiting!
